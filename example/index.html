Базовые понятия

Взаимодействие с апи ккм сервером подразделяется на следуюшие подзадачи

  • обработку ответа сервера
  • формирование запроса к нему
Для начала работы нужно подключить библиотеку
<script src="kkmserver.js"></script>
Создаем объект для взаимодействия с сервером и назначаем обработчик
Kkm = new KkmServer('пользователь', 'пароль','урл сервера').hookAjaxSuccess('ваша функция');
